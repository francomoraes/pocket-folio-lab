# InvestTracker Frontend

Aplicacao web para acompanhamento e gestao de investimentos.

## Requisitos

- Node.js 20+
- npm 10+

## Scripts

- `npm run dev`: inicia ambiente local em modo desenvolvimento.
- `npm run build`: gera build de producao.
- `npm run preview`: sobe a build localmente para validacao.
- `npm run lint`: executa verificacao de lint.

## Desenvolvimento local

```sh
npm install
npm run dev
```

## Stack

- React + TypeScript
- Vite
- Tailwind CSS
- shadcn/ui
